When you find yourself having a minute to yourself, you pick up your smartphone and start playing. The games need to be short, engaging and competitive, for it to have real success. An addictive game, Speed Reactor aims to fulfill that need. It is a game that rewards the player based on his ability to react quickly, in a few scenarios.

The flow of the app is:

The user registers (using Google Game Play Services).
The user selects the type of game he wishes to play (Fast game, Challenge, Zen).
The user selects the puzzles he wishes to play.
The user plays the game.
The user is awarded achievements and perks based on the gameplay.
The user is placed in a LeaderBoard.

The app works on Android.
